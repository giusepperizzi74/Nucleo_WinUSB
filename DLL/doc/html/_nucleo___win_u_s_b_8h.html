This function configures the output type (0-pushpull, 1-opendrain) and the pull-up/down resistors (0-none, 1-up, 2-down) of a GPIO pin.

Parameters
[in]handleHandle obtained from open().
[in]portGPIO port. Must be a letter from 'a' to 'h'.
[in]pinGPIO pin. Must be a number from 0 to 15.
[in]direction.Must be 0 for input, 1 for output.
[in]typeOutput type. Must be 0 for pushpull, 1 for opendrain.
[in]pullPullup/down resistors. Must be 0 for none, 1 for pullup and 2 for pulldown.
Returns
int variable. Holds the operation result [success(>=0), fail(<0)]

NUCLEO_WINUSB_API void * open ( char * desc)

Gets a USB device handle from its descriptor.

Use this function to get a handle to a specific device from its descriptor, which can be obtained with get_next(). The device default interface is opened.
